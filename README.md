# Python Socket Chat Application
This is a real-time chat application built with Python's socket programming library that demonstrates how computer networks enable communication between different devices.

## Features
The app consists of two components:

A central server that manages all connections

Multiple clients that users run to join the chat

## How to Run

1. Save files - chat_server.py and chat_client.py

2. Open terminal in file location

3. Start server: python chat_server.py

4. Start client(s): python chat_client.py (new terminal)

5. Enter username when prompted

6. Start chatting
7. Server stop: In server terminal type shutdown
